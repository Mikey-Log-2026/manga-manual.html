# MiKey Log - フォークリフト貸出管理システム

MiKey Logを活用した、現場のDX（デジタルトランスフォーメーション）事例を紹介します。

## 📋 導入の背景と解決策
これまで有人窓口で行っていたフォークリフトの鍵貸出業務を、**MiKey Log**によって自動化しました。

| 項目 | 導入前（Before） | 導入後（After / MiKey Log） |
| :--- | :--- | :--- |
| **貸出方法** | 担当窓口へ行き、対面で借りる | ヘルメットのQRをスキャンして自動解錠 |
| **待ち時間** | 担当者不在時は作業がストップ | 24時間いつでも数秒で貸出可能 |
| **記録管理** | 手書き台帳（紛失のリスク） | ダッシュボードへ自動保存（リアルタイム可視化） |
